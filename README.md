# AIRLINE-FARE-SELECTION-BEHAVIOUR-ANALYSIS

## PROJECT OVERVIEW
 This analysis examines Airline passenger fare selection patterns to evaluate the effectiveness of the airline’s upsell strategy. It explores how factors such as trip type, device category, and traveller group influence customers’ decisions to upgrade from the default fare to higher-priced options. The goal is to identify behavioral trends, measure upsell conversion performance, and uncover opportunities to optimize pricing, targeting, and revenue growth.

### DATA SOURCES

Airline_fare Upsell performance Data : The Primary dataset used for the analysis is the Airline_Fare_Upsell_performance.csv file containing the major data for the analysis.
